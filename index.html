What is the Contribution Unit System?
The Contribution Unit System recognizes and tracks members' contributions throughout their journey with Sincerely 343. These "units" are not about competition or status — many members will be contributing in different ways and it is important to be transparent about visibility and fairness.
Not a points game
Units encourage consistency and awareness, not comparisons to other members.
Not the barter system
Barter is peer-to-peer. CUs measure participation within the 343 ecosystem as a whole.
Transparency first
Some months more, some less — that's okay. Wellness and balance come first.
How points work
1 pt Lighter work, ideally under an hour — attending a session, donating a cord, giving feedback.
2 pts Moderate work, 3–5 hours — editing, sound design, hosting an event.
3 pts Heavy lift, multiple days or ongoing — supervising a project, full orientation, high-value gear.
Points reflect meaningful effort, not time spent. A 30-minute mentorship that changes someone's direction is as valuable as an all-day shoot.
Key Rules
• No public ranking — totals stay private
• Points only accumulate, you can never lose them
• Members report contributions outside official channels
• Disputes can be handled immediately
• Higher points ≠ more authority — leadership is earned through trust

How to get your token:
1. Go to developers.facebook.com
2. Create an app → Add Instagram Graph API product
3. Generate a long-lived access token with instagram_basic, instagram_manage_insights scopes
4. Find your User ID in the Graph API Explorer
